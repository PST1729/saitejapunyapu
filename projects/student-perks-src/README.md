# StudentPerks

Search 400+ verified student discounts across AI tools, dev platforms, streaming, learning, and more. Free, no signup.

## Development

Requires Node.js 18+.

```sh
cd projects/student-perks-src
npm install
npm run dev
```

Dev server: `http://localhost:5173/projects/student-perks/`

## Build

```sh
cd projects/student-perks-src
npm run build
```

Static output is written to `projects/student-perks/` (served at `/projects/student-perks/` on the portfolio).

## Tech stack

- React 19
- Vite
- TypeScript
- Tailwind CSS v4
